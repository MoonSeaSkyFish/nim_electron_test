# Nim Electron read file example.

Nim + Electron read file example.

## How to
```
% git clone https://github.com/MoonSeaSkyFish/nim_electron_test.git
% cd nim_electron_test/fs
% npm install
% nimble build
% nimble start
```

Or:

```
% git clone https://github.com/MoonSeaSkyFish/nim_electron_test.git
% cd nim_electron_test/fs
% npm install
% nim js index.nim
% nim js preload.nim
% nim js renderer.nim
% npx electron .
```
