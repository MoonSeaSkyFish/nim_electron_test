# Nim Electron Quickstart

Nim + Electron Quik Start.

## How to
```
% git clone https://github.com/MoonSeaSkyFish/nim_electron_test.git
% cd nim_electron_test/quickstart
% npm install
% nimble build
% nimble start
```

Or:

```
% git clone https://github.com/MoonSeaSkyFish/nim_electron_test.git
% cd nim_electron_test/quickstart
% npm install
% nim js index.nim
% npx electron .
```

Display "Hello, Electron !" in window, sucessed.
